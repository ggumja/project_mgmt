# B2B 쇼핑몰 기능 리스트
## 기본 쇼핑몰 기능 + B2B 특화 기능

---

## 📦 1. 기본 쇼핑몰 기능 (B2C 공통)

### 1.1 회원 관리
- [ ] **회원가입/로그인**
  - 이메일/비밀번호 인증
  - 소셜 로그인 (Google, Kakao, Naver)
  - 비밀번호 찾기/재설정
  - 이메일 인증

- [ ] **회원 정보 관리**
  - 프로필 수정
  - 비밀번호 변경
  - 배송지 관리 (다중 배송지)
  - 회원 탈퇴

- [ ] **마이페이지**
  - 주문 내역 조회
  - 배송 조회
  - 찜 목록
  - 쿠폰함
  - 포인트 내역

### 1.2 상품 관리
- [ ] **상품 등록/수정/삭제**
  - 상품명, 설명, 가격
  - 상품 이미지 (다중 이미지)
  - 상품 옵션 (색상, 사이즈 등)
  - 재고 수량
  - 상품 상태 (판매중/품절/판매중지)

- [ ] **카테고리 관리**
  - 대/중/소 카테고리 구조
  - 카테고리별 상품 분류
  - 카테고리 순서 관리

- [ ] **상품 검색/필터링**
  - 키워드 검색
  - 카테고리별 필터
  - 가격대별 필터
  - 정렬 (인기순, 최신순, 가격순)

- [ ] **상품 상세 페이지**
  - 상품 정보 표시
  - 이미지 갤러리
  - 옵션 선택
  - 수량 선택
  - 리뷰/평점
  - 상품 문의

### 1.3 장바구니
- [ ] **장바구니 관리**
  - 상품 추가/삭제
  - 수량 변경
  - 옵션 변경
  - 선택 상품 주문
  - 전체 상품 주문

- [ ] **장바구니 기능**
  - 비회원 장바구니 (쿠키)
  - 회원 장바구니 (DB 저장)
  - 품절 상품 알림
  - 가격 변동 알림

### 1.4 주문/결제
- [ ] **주문서 작성**
  - 주문자 정보
  - 배송지 정보
  - 배송 메모
  - 결제 수단 선택

- [ ] **결제 처리**
  - 신용카드
  - 계좌이체
  - 무통장입금
  - 간편결제 (카카오페이, 네이버페이, 토스)
  - 포인트/쿠폰 사용

- [ ] **주문 관리**
  - 주문 내역 조회
  - 주문 상태 추적
  - 주문 취소
  - 교환/반품 신청

### 1.5 배송 관리
- [ ] **배송 정보**
  - 배송 상태 (준비중/배송중/배송완료)
  - 송장 번호 조회
  - 배송 추적

- [ ] **배송 옵션**
  - 일반 배송
  - 빠른 배송
  - 새벽 배송
  - 택배사 선택

### 1.6 리뷰/평점
- [ ] **리뷰 작성**
  - 텍스트 리뷰
  - 별점 평가
  - 이미지 첨부
  - 구매 확정 후 작성

- [ ] **리뷰 관리**
  - 리뷰 수정/삭제
  - 리뷰 신고
  - 베스트 리뷰
  - 포토 리뷰

### 1.7 고객 지원
- [ ] **1:1 문의**
  - 상품 문의
  - 주문 문의
  - 배송 문의
  - 기타 문의

- [ ] **FAQ**
  - 자주 묻는 질문
  - 카테고리별 FAQ

- [ ] **공지사항**
  - 공지사항 목록
  - 중요 공지 상단 고정

### 1.8 프로모션/마케팅
- [ ] **쿠폰**
  - 쿠폰 발행
  - 쿠폰 다운로드
  - 쿠폰 사용

- [ ] **포인트**
  - 포인트 적립
  - 포인트 사용
  - 포인트 내역

- [ ] **이벤트**
  - 기획전
  - 타임세일
  - 할인 이벤트

## 🏢 2. B2B 특화 기능

### 2.1 B2B 회원 관리
- [ ] **사업자 회원가입**
  - 사업자등록번호 입력/인증
  - 사업자등록증 업로드
  - 회사 정보 (상호, 대표자명, 업종)
  - 담당자 정보 (이름, 연락처, 이메일)
  - 승인 대기 상태

- [ ] **회원 승인 프로세스**
  - 관리자 승인 대기
  - 사업자 정보 검증
  - 승인/반려 처리
  - 승인 알림 (이메일/SMS)

- [ ] **회원 등급 관리**
  - VIP/골드/실버/일반 등급
  - 등급별 혜택 (할인율, 배송비)
  - 등급 자동 산정 (구매 금액 기준)
  - 등급별 상품 노출

- [ ] **다중 사용자 계정**
  - 회사당 여러 사용자 등록
  - 사용자별 권한 설정 (구매자/승인자/관리자)
  - 부서별 계정 관리

### 2.2 B2B 상품 관리
- [ ] **대량 상품 등록**
  - 엑셀 파일 업로드
  - CSV 파일 업로드
  - 상품 일괄 수정
  - 이미지 일괄 업로드

- [ ] **B2B 가격 정책**
  - 기본 판매가
  - 회원 등급별 가격
  - 수량별 할인 (10개 이상, 50개 이상 등)
  - 거래처별 특별 가격
  - 시즌별 가격 정책

- [ ] **최소 주문 수량 (MOQ)**
  - 상품별 최소 주문 수량 설정
  - 박스/케이스 단위 주문
  - 혼합 주문 가능 여부

- [ ] **상품 카탈로그**
  - PDF 카탈로그 다운로드
  - 상품 스펙 시트
  - 대량 주문 가이드

### 2.3 B2B 주문/결제
- [ ] **견적서 시스템**
  - 견적서 요청
  - 견적서 발행
  - 견적서 승인/반려
  - 견적서 유효기간
  - 견적서 PDF 다운로드

- [ ] **승인 워크플로우**
  - 구매 요청 → 승인자 검토 → 주문 확정
  - 다단계 승인 (부서장 → 임원)
  - 승인 권한 설정
  - 승인 알림

- [ ] **대량 주문**
  - 엑셀 파일로 주문
  - 장바구니 대량 추가
  - 이전 주문 재주문
  - 정기 주문 설정

- [ ] **결제 방식**
  - 외상 거래 (후불)
  - 선불 결제
  - 월말 결제
  - 어음 결제
  - 신용 한도 관리

- [ ] **발주서 관리**
  - 발주서 발행
  - 발주서 번호 자동 생성
  - 발주서 PDF 다운로드
  - 발주 내역 조회

### 2.4 정산/세금계산서
- [ ] **세금계산서 자동 발행**
  - 주문 확정 시 자동 발행
  - 국세청 전자세금계산서 연동
  - 세금계산서 이메일 발송
  - 세금계산서 PDF 다운로드

- [ ] **정산 관리**
  - 월별 정산 내역
  - 미수금 관리
  - 입금 확인
  - 정산서 발행

- [ ] **거래 명세서**
  - 거래 명세서 발행
  - 월별 거래 내역
  - 거래처별 집계

### 2.5 재고/물류 관리
- [ ] **실시간 재고 확인**
  - 상품별 재고 수량
  - 창고별 재고 현황
  - 재고 부족 알림
  - 입고 예정 정보

- [ ] **대량 배송**
  - 팔레트 배송
  - 화물 배송
  - 직접 픽업
  - 배송 일정 조율

- [ ] **ERP 연동**
  - 주문 데이터 자동 전송
  - 재고 동기화
  - 출고 지시
  - 배송 정보 업데이트

### 2.6 고객 관리 (CRM)
- [ ] **거래처 관리**
  - 거래처 정보 관리
  - 거래 이력 조회
  - 담당자 관리
  - 거래처별 메모

- [ ] **구매 패턴 분석**
  - 자주 구매하는 상품
  - 구매 주기 분석
  - 재구매율
  - 추천 상품

- [ ] **맞춤 서비스**
  - 전담 영업 담당자 배정
  - 1:1 상담
  - 맞춤 견적
  - 특별 요청 처리

### 2.7 계약/협약 관리
- [ ] **공급 계약**
  - 계약서 업로드
  - 계약 기간 관리
  - 계약 조건 (가격, 수량, 납기)
  - 계약 갱신 알림

- [ ] **장기 거래 할인**
  - 연간 계약 할인
  - 물량 약정 할인
  - 독점 공급 계약

### 2.8 리포트/분석
- [ ] **구매 분석**
  - 월별/분기별 구매 통계
  - 상품별 구매 현황
  - 부서별 구매 내역
  - 예산 대비 지출

- [ ] **재주문 알림**
  - 재고 소진 예측
  - 자동 재주문 제안
  - 정기 구매 알림

- [ ] **대시보드**
  - 주문 현황
  - 배송 현황
  - 미수금 현황
  - 주요 지표 (KPI)

### 2.9 API 연동
- [ ] **구매 시스템 연동**
  - 고객사 ERP 연동
  - 구매 시스템 API
  - 재고 조회 API
  - 주문 생성 API

- [ ] **물류 시스템 연동**
  - WMS (창고관리시스템) 연동
  - 배송 추적 API
  - 입출고 관리

### 2.10 모바일 B2B
- [ ] **모바일 주문**
  - 반응형 웹
  - 모바일 앱 (선택사항)
  - 바코드 스캔 주문
  - 음성 주문

- [ ] **영업 사원용 기능**
  - 고객 방문 기록
  - 현장 주문 접수
  - 재고 확인
  - 견적서 즉시 발행

## 🎯 3. 우선순위별 구현 계획

### Phase 1: MVP (필수 기능) - 8주
**기본 쇼핑몰 기능**
- ✅ 회원가입/로그인
- ✅ 상품 목록/상세
- ✅ 장바구니
- ✅ 주문/결제
- ✅ 관리자 상품 관리

**B2B 핵심 기능**
- ✅ 사업자 회원가입/승인
- ✅ 회원 등급별 가격
- ✅ 견적서 시스템
- ✅ 세금계산서 발행

### Phase 2: 고급 기능 - 12주
**기본 쇼핑몰 확장**
- 리뷰/평점
- 쿠폰/포인트
- 1:1 문의

**B2B 확장 기능**
- 승인 워크플로우
- 대량 주문
- 정산 관리
- 거래처 관리

### Phase 3: 최적화 - 16주
**고급 B2B 기능**
- ERP 연동
- API 제공
- 구매 패턴 분석
- 모바일 앱
